
# Customer Churn Prediction
Machine Learning project to predict customer churn using behavioral and demographic data.

## Summary
- Built and evaluated predictive models (Logistic Regression, Random Forest).  
- Identified key churn drivers such as contract type and payment method.  
- Achieved 86% accuracy and 0.91 ROC-AUC on test data.

## Tools
Python, Pandas, Scikit-learn, Matplotlib, Seaborn
